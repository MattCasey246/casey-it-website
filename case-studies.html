Case Studies

We have delivered infrastructure and cloud transformation projects for a range of organisations across the UK. The following case studies illustrate the types of engagements we undertake and the outcomes we achieve.

Multi-Site Infrastructure Transformation

Residential Property / Build-to-Rent 2023 – Present (Ongoing) Contract Infrastructure Consultant London & South East

The Challenge

Disparate, legacy network and server estate across multiple residential sites creating operational inconsistency, poor visibility, and growing security risk.

What We Delivered

  • Full network standardisation migrating Fortinet/HP/Aruba estates to Cisco Meraki
  • Infrastructure remediation migrating legacy Microsoft server estate to Azure Public Cloud
  • IT mobilisation for four new neighbourhood launches (Maidenhead, Leatherhead, Lewisham, Birmingham) — 1,800+ residential units
  • Microsoft 365 and Azure AD security remediation following audit findings
  • Full asset discovery and centralised infrastructure inventory
Outcome

A standardised, cloud-first infrastructure estate with improved security posture and a scalable platform for ongoing portfolio growth.

New Site IT Mobilisation — Premium Retirement Living

Retirement Living / Hospitality July 2022 – April 2023 Network Manager / Infrastructure & Cloud Consultant

The Challenge

New premium retirement living site requiring complete IT infrastructure design and delivery ahead of opening.

What We Delivered

  • Full network stack: leased line, Meraki firewall, VLAN segmentation, VPNs, WiFi, and VoIP
  • Apple Business Manager rollout with device lifecycle management
  • Azure, Intune, and conditional access policies
  • Microsoft licensing cost reduction of 12% through audit and optimisation
  • Penetration testing coordination and security readiness
Outcome

Fully operational, secure IT environment delivered on time for site opening with measurable reduction in licensing spend.

Infrastructure Remediation — Social Housing

Social Housing / Public Sector July 2020 – June 2022 Infrastructure Consultant

The Challenge

Ageing 500-server estate with significant security vulnerabilities, legacy software, and no centralised inventory.

What We Delivered

  • Full audit of 500-server estate with master inventory and remediation roadmap
  • Cisco Meraki WiFi across 15 sites within a 3-week delivery window
  • Windows Server upgrade programme (2003/2016 to 2019), modernising AD, DNS, DHCP
  • Penetration test vulnerability remediation and strengthened AD security model
Outcome

Dramatically improved security posture, modernised server estate, and standardised network across all 15 sites.
